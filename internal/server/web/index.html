    About

    Version: …

    Network Monitor is a self-contained Windows application for tracking internet connectivity over time. It runs ping and traceroute probes on a configurable schedule, records results in a local SQLite database, and presents them through a browser-based dashboard — including latency history, route visualization, and speed tests. No installation, no cloud services, no external dependencies.
